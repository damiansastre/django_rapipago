# django_rapipago


## Requirements

* Python >=3

## Installation

```
git clone https://github.com/tagercito/django_rapipago
cd django_rapipago
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

## Tests
We use django's test suit for testing

```
python manage.py test
```
